## Descrição
O projeto utiliza Flexbox para organizar os elementos da página em uma única coluna, criando a aparência de um livro antigo. Foram utilizadas cores inspiradas em papel envelhecido, tipografia serifada e elementos visuais simples para reforçar a temática de livro.
## Recursos Utilizados
- CSS Flexbox
- Pseudo-elementos (::before e ::after)
- Tipografia serifada
- Media Queries
- Sombras e bordas decorativas
