# Projeto — Recriação Responsiva

## Autor
Adonai Roberto de Oliveira Narciso

## Site Escolhido
Netflix

## Link do Site
https://www.netflix.com/br/

## Objetivo
Criar uma versão responsiva inspirada na homepage da Netflix.

## Estratégia Responsiva
Mobile-first utilizando Flexbox e CSS Grid.

## Breakpoints
- 768px
- 1024px
- 1440px

## Recursos Utilizados
- CSS Grid
- Flexbox
- clamp()
- prefers-color-scheme
- Media Queries

## Adaptações Realizadas
O layout foi simplificado para melhor adaptação mobile, mantendo apenas os principais elementos da homepage.

## Dificuldades
Implementação do hero com imagem de fundo e responsividade dos cards.
